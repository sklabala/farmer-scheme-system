# Deliverable 5: User Manual

## 1. Introduction

### 1.1 Purpose

The Farmer Scheme Information System (FSIS) is a digital platform designed to help Indian farmers discover, understand, and apply for government agricultural schemes. This manual provides step-by-step instructions for farmers to use the system effectively.

### 1.2 Target Users

- Small and marginal farmers
- Large farmers seeking scheme benefits
- Agricultural laborers
- Farm women

### 1.3 Supported Languages

- English (EN)
- Hindi (HI)
- Tamil (TA)
- Telugu (TE)
- Kannada (KN)
- Marathi (MR)

## 2. Getting Started

### 2.1 Registration

#### Step 1: Download and Install
- Visit Google Play Store or Apple App Store
- Search "Farmer Scheme Portal"
- Download and install the application

#### Step 2: First Launch
1. Open the app
2. Select your preferred language
3. Click "Register with Phone Number"

#### Step 3: Enter Details
```
┌─────────────────────────────────────┐
│         Phone Number                │
│  [ +91 ] [ 9 1 2 3 4 5 6 7 8 9 ]   │
│                                     │
│         Full Name                   │
│  [ ________________________________]│
│                                     │
│         State                       │
│  [ Select your state           ▼ ]  │
│                                     │
│         District                    │
│  [ Select your district        ▼ ]  │
│                                     │
│        [ VERIFY WITH OTP ]          │
└─────────────────────────────────────┘
```

#### Step 4: OTP Verification
1. You will receive a 6-digit OTP via SMS
2. Enter the OTP in the verification field
3. Click "Verify"

#### Step 5: Profile Setup (Optional)
- Add your farming details
- Set preferred crops
- Update land size
- Add irrigation method

### 2.2 Login

#### Quick Login (OTP)
1. Enter your registered phone number
2. Click "Get OTP"
3. Enter the OTP received
4. Click "Login"

#### Forgot Phone Number?
Contact your nearest Common Service Centre (CSC) for assistance.

### 3.3 Eligibility Check

Before applying, check if you qualify:

1. Open the scheme details page
2. Tap "Check Eligibility"
3. The system will analyze your profile
4. View the eligibility result:
   - ✅ "You are eligible!"
   - ❌ "Not eligible due to..."

### 3.5 Tracking Applications

#### View Application Status
1. Go to "My Applications" from menu
2. View all your applications
3. Tap on any application for details

#### Status Meanings
| Status | Meaning |
|--------|---------|
| 🟡 Pending | Application submitted, under review |
| 🔵 Under Review | Being examined by officials |
| 🟢 Approved | Application accepted |
| 🔴 Rejected | Not approved (see reasons) |
| ⚪ Closed | Process completed |

### 3.6 Notifications

#### Notification Types
- 📢 **New Schemes** - New schemes matching your profile
- ⏰ **Deadline Reminders** - Upcoming application deadlines
- 📊 **Status Updates** - Changes in application status
- 🌾 **Weather Alerts** - Weather warnings for your region

#### Managing Notifications
1. Go to Settings → Notifications
2. Choose what alerts to receive
3. Select communication channel (SMS/Push/WhatsApp)

## 5. Help & Support

### 5.1 Getting Help

#### In-App Help
- Tap "Help" in the menu
- Search for your question
- Browse FAQ topics

#### Contact Support
- Phone: 1800-XXX-XXXX (Toll-free)
- Email: support@farmerscheme.gov.in
- Hours: 9 AM - 6 PM (Mon-Sat)

#### Offline Help
- Visit nearest Common Service Centre (CSC)
- Meet your Block Agriculture Officer
- Visit Gram Panchayat office

### 5.2 Common Issues

| Problem | Solution |
|---------|----------|
| OTP not received | Check network, resend after 30 seconds |
| Application rejected | View reasons, reapply with corrections |
| Document upload failed | Check file size (<2MB), try again |
| Can't find scheme | Use search or change filters |
| App running slow | Clear cache, restart app |

## 6. Safety Tips

### 6.1 Security

🔒 **Keep your account safe:**
- Never share your OTP with anyone
- Do not use public WiFi for login
- Log out after using shared devices
- Report suspicious messages

### 6.2 Avoiding Scams

⚠️ **Important Notices:**
- Government schemes are FREE
- Never pay money to apply
- No agent needed for online applications
- Official communication only from @farmerscheme.gov.in
